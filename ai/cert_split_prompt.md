# 证书拆分AI提示词文档

## 1. 核心指令
```
请根据以下规则，将提供的证书文本拆分为标准化的证书组合列表
```

## 2. 输入数据特征
- 多本证书用连接符组合
- 常见连接符：`+`、`加`、`带`、`及`、`和`
- 可能存在简写、别名、口语化表达

## 3. 标准化映射规则

### 3.1 建造师证书
```
一级建造师 → 一级建造师
二级建造师 → 二级建造师
一建 → 一级建造师
二建 → 二级建造师
```

### 3.2 专业类别
```
建筑 → 建筑工程
房建 → 建筑工程
土建 → 建筑工程
市政 → 市政公用工程
水利 → 水利水电工程
机电 → 机电工程
公路 → 公路工程
```

### 3.3 职称证书
```
中级职称 → 中级职称
中工 → 中级工程师
高工 → 高级工程师
助工 → 助理工程师
```

### 3.4 安全证书
```
A → A证
B → B证
C → C证
ABC → ABC证
安全员A/B/C → 安全员A证/B证/C证
```

### 3.5 其他证书
```
造价 → 造价工程师
监理 → 监理工程师
结构 → 结构工程师
建筑 → 建筑师
```

## 4. 输出格式要求
```python
# 标准输出格式
['证书1', '证书2', '证书3', ...]
```

## 5. 完整提示词模板
```
请按照以下规则处理证书文本：

1. 拆分规则：
   - 主要分隔符：+、加、带、及、和
   - 保持证书组合的完整性
   - 每个证书组合用方括号表示

2. 标准化映射：
   {此处插入具体的映射规则}

3. 输出要求：
   - 返回Python一维列表格式
   - 证书名称使用标准化全称
   - 按照输入顺序排列证书

请处理以下证书文本：
【用户输入的证书文本】
```

## 6. 示例演示

### 输入：
```
一级房建+二级市政+中工+ABC
```

### 输出：
```python
['一级建造师建筑工程', '二级建造师市政公用工程', '中级工程师', 'ABC证']
```

## 7. 特殊处理规则

### 7.1 专业合并
```
一级建造师建筑 → 一级建造师建筑工程
二级市政 → 二级建造师市政公用工程
```

### 7.2 证书组合
- 建造师+专业：合并为完整证书名称
- 职称+等级：保持原样
- 安全证书：统一添加"证"字

### 7.3 去重规则
- 同一组合中重复证书去重
- 不同组合保持独立

## 8. 错误处理
- 无法识别的证书：保持原样输出
- 格式错误：尝试智能修正
- 歧义表达：选择最常见含义

## 9. 使用说明
将此提示词文档保存，在处理证书拆分任务时：
1. 根据具体需求调整映射规则
2. 插入用户的实际证书文本
3. 保持输出格式的一致性

这个提示词文档可以确保AI在处理各类证书组合时，能够准确拆分并标准化输出。
